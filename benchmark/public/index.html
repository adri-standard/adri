Benchmark Overview

About the Benchmark

The Agent Data Readiness Index (ADRI) Benchmark provides insights into how well different industries and data sources communicate their quality attributes to AI agents. This benchmark is based on anonymized assessment results from organizations across various sectors.

The benchmark is updated quarterly as more organizations contribute their assessment results. All data is anonymized to protect confidentiality.

Contribute Your Results

Help improve the benchmark by contributing your anonymized assessment results. You can run the ADRI assessment tool on your data sources and submit the results to be included in the benchmark.

To contribute, run the following command after completing your assessment:

adri benchmark submit path/to/report.json --industry "Your Industry" --anonymize

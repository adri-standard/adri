ADRI Audience Badges Example

This page demonstrates how to use the audience badges in your documentation.

Inline Audience Badges

You can use audience badges inline in your content:

🤖 AI Builder 📊 Data Provider 🛠️ Standard Contributor

Audience-Specific Headings

You can add audience indicators to headings using the data-audience attribute:

This heading is for AI Builders

Content specifically for developers building AI applications and agents.

This heading is for Data Providers

Content specifically for teams that manage, prepare, and deliver data for AI systems.

This heading is for Standard Contributors

Content specifically for open source contributors and organizations improving ADRI.

Usage in Documentation

To use these badges in your documentation:

For inline badges:

<span class="audience-badge audience-badge-ai-builder"><span class="emoji">🤖</span> AI Builder</span>

For heading badges:

<h3 data-audience="AI_BUILDER">This heading is for AI Builders</h3>

Available audience types:

  • AI_BUILDER - For developers building AI applications and agents
  • DATA_PROVIDER - For teams that manage, prepare, and deliver data for AI systems
  • STANDARD_CONTRIBUTOR - For open source contributors and organizations improving ADRI

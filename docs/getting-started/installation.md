# Installation Guide

## Requirements

- Python 3.8+
- pip or conda

## Install from PyPI

```bash
pip install adri
```

## Install from Source

```bash
git clone https://github.com/adri-standard/adri.git
cd adri
pip install -e .
```

## Verify Installation

```python
<!-- audience: ai-builders -->
import adri
print(adri.__version__)
```
